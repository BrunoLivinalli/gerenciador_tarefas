# Gerenciador de Tarefas em Python  
Um projeto simples e funcional desenvolvido em **Python**, com foco em consolidar fundamentos de lógica, estruturas de dados, modularização e interação via terminal.  
O objetivo é demonstrar domínio dos conceitos iniciais **Back-End**.

---

## Funcionalidades

- Adicionar novas tarefas  
- Listar tarefas existentes  
- Marcar tarefas como concluídas  
- Remover tarefas  
- Persistência simples de dados (arquivo `.txt` ou `.json`)  
- Menu interativo no terminal  

---

## Objetivo do Projeto

- Treinar lógica e fundamentos de Python  
- Praticar manipulação de listas e dicionários  
- Aplicar funções e modularização  
- Criar uma aplicação simples mas completa  

---

## Tecnologias utilizadas
- **Python 3.x**  
- Manipulação de arquivos (`open`)  
- Estruturas de dados padrão
